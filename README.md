# 📱 Mobile Sales Dashboard | Power BI

An interactive **Power BI Mobile Sales Dashboard** designed to analyze sales performance, customer behavior, payment methods, and business trends. This dashboard provides actionable insights through dynamic KPIs, interactive visualizations, and time intelligence analysis.

---

## 📌 Project Overview

The **Mobile Sales Dashboard** was built using **Power BI** to help businesses monitor and analyze mobile sales performance. It provides a comprehensive overview of key business metrics, allowing users to identify sales trends, compare historical performance, evaluate customer satisfaction, and make data-driven decisions.

The dashboard includes executive KPIs, monthly and quarterly sales analysis, brand and product performance, payment method analysis, customer rating insights, and Year-over-Year comparisons using **Month-to-Date (MTD)** and **Same Period Last Year (SPLY)** calculations.

---

## 🎯 Business Objectives

- Monitor overall sales performance.
- Track Total Sales, Quantity Sold, Transactions, and Average Selling Price.
- Compare current sales with the Same Period Last Year (SPLY).
- Analyze Month-to-Date (MTD) performance.
- Identify top-performing mobile brands and models.
- Understand customer purchasing behavior.
- Analyze payment method preferences.
- Monitor customer satisfaction using rating analysis.
- Discover monthly, quarterly, and daily sales trends.

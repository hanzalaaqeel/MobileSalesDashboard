## 📌 Overview

The **FinSight Dashboard** is an end-to-end Business Intelligence project developed in **Power BI** to transform raw financial transaction data into actionable business insights. It provides an executive-level overview of financial performance through interactive dashboards, helping organizations monitor revenue, transaction activity, customer behavior, and operational trends.

The dashboard combines **Power Query**, **Data Modeling**, **DAX Measures**, and modern visualization techniques to create an intuitive reporting solution suitable for finance professionals, analysts, and business decision-makers.

---

## 🎯 Business Objectives

- Monitor overall financial performance.
- Track total transaction amount and transaction volume.
- Analyze average transaction value.
- Evaluate transaction fees and tax collection.
- Identify successful, pending, and failed transactions.
- Understand customer segmentation and demographics.
- Analyze revenue by transaction type.
- Compare financial performance across states.
- Enable faster, data-driven business decisions.
